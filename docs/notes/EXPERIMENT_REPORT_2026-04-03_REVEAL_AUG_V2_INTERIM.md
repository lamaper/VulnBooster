# reveal_aug_v2 中期评估

更新时间：2026-04-03

## 1. 当前流程状态

本轮 `reveal_aug_v2` 大规模增强实验尚未全部结束。

当前已经完成的阶段：

1. DeepSeek 并发生成
2. `post_process.py`
3. `post_analysis.py`
4. 增强任务 `reveal_aug_v2` 构建
5. `origin_s` guide 模型训练

当前正在执行的阶段：

1. 基于 formal guide 模型重生成 `linevul-CausalCode-token` 配对域

尚未完成的阶段：

1. 最终双域训练
2. 在 `devign` 上的最终测试

因此，当前能够报告的是“增强数据质量与规模”和“guide 阶段表现”，还不能把它误认为整轮最终跨域结果。

## 2. 增强数据结果

生成与过滤结果如下：

- 处理种子数：`200`
- 每个种子目标变体数：`4`
- 生成成功 seed 数：`195`
- resume 跳过 seed 数：`5`
- 提取出的有效漏洞变体：`1324`
- 相似度过滤后保留：`1321`

增强任务构建结果如下：

- 原始训练样本：`18188`
- 原始测试样本：`4546`
- 新增增强样本：`1307`
- 最终训练样本：`19495`
- 最终测试样本：`4546`

与上一轮 `reveal_aug_v1` 相比：

- `reveal_aug_v1` 新增增强样本：`4`
- `reveal_aug_v2` 新增增强样本：`1307`

这说明本轮已经不再是“闭环验证级别”的增强，而是进入了真正可用于正式实验的大规模版本。

## 3. guide 模型结果

当前已完成的 guide 模型训练结果如下：

- Epoch 3：`Acc 91.46% | P 83.42% | R 53.61% | F1 65.27%`
- Epoch 4：`Acc 89.67% | P 66.42% | R 62.76% | F1 64.54%`
- Epoch 5：`Acc 91.63% | P 81.91% | R 56.40% | F1 66.80%`

guide 阶段最佳验证 F1 为：

- `66.80%`

## 4. 中期分析

当前可以确认的正向结论有三点：

1. 并发生成方案有效，将生成阶段压缩到了可接受时长，并成功产出了大规模增强数据。
2. `reveal_aug_v2` 的增强规模相较 `reveal_aug_v1` 有数量级提升。
3. 在增强后的训练任务上，guide 模型在验证集上的 F1 已达到 `66.80%`，显著高于此前小规模版本的 guide 表现。

但当前仍需强调：

- `66.80%` 是 `reveal_aug_v2` 训练内验证结果
- 不是最终 `reveal_aug_v2 -> devign` 跨域测试结果
- 最终论文结论仍然要看“配对域重生成完成后”的双域测试表现

## 5. 当前最合理的阶段判断

目前最合理的判断是：

- 大规模 LLM 增强已经成功落地
- 增强数据量已经足以支持一轮正式实验
- guide 阶段信号明显优于上一轮小样本版本
- 但是否真正转化为跨域 `devign` F1 提升，还要等最终双域测试结束
